# 课程设计

## IEEE754格式

- S，符号位：1
- E，指数位：11
- F，尾数位：52

### 规格化数

指数不全为零也不全为一（1~2046）

计算公式：```V = (-1)^S × (1 + F) × 2^(E - 1023)```

### 特殊值

指数全为零或全为一

#### 非规格化数

非常接近于0的数：指数全为零，尾数不为零

计算公式：```V = (-1)^S × (0 + F) × 2^(-1022)```

#### 零

指数位全为零，尾数位也全为零

分正负零

#### 无穷大

表示计算结果上溢，如`1.0/0.0`：指数位全为一，尾数位全为零

分正负无穷

#### 非数值

无效或未定义的计算结果，如`0.0/0.0`或`sqrt(-1.0)`：指数位全为一，尾数位不为零

静默NaN：尾数位最高位为1，任何涉及qNaN的运算，结果仍是qNaN

信号NaN：尾数位最高位为0，sNaN触发“无效操作”的浮点异常，可被程序捕获，通常用于标记未初始化的变量或支持更高级的计算特性

## 舍入方式

### 平衡舍入

向最接近的偶数舍入

## 计算

- **任何涉及 NaN 的运算**：结果一律为 NaN (例如 `5 + NaN` -> `NaN`)。
- 无穷大 (∞) 的运算：
  - `∞ + ∞ = ∞`
  - `∞ - ∞ = NaN` (未定义)
  - `∞ × c = ∞` (c 为任意正数)
  - `∞ ÷ c = ∞`
  - `c ÷ ∞ = 0`
  - `∞ ÷ ∞ = NaN` (未定义)
  - `∞ × 0 = NaN` (未定义)
- 零 (0) 的运算：
  - `x + 0 = x`
  - `x - 0 = x`
  - `x × 0 = 0`
  - `x ÷ 0 = ±∞` (符号取决于 x 的符号，x≠0)
  - `0 ÷ 0 = NaN` (未定义)

### 加减法

1. 拆分为符号位(S_A, S_B)、指数位(E_A, E_B)和尾数位(F_A, F_B)
2. 为尾数位加上隐藏位，规格化数是1，非规格化数是0，得到完整的(M_A, M_B)
3. 对齐指数
   1. 获取指数差d
   2. 指数较小的那个数的尾数向右移动d位
   3. 结果指数设置为较大的指数
4. 符号相同，则将对齐后的尾数相加
   符号不同，则将较大尾数减去较小尾数，结果符号与较大尾数的那个数一致
5. 规格化
   1. 尾数整数部分大于一则将指数+1，直到尾数整数部分为1
   2. 尾数整数部分为0，则指数-1，直到尾数整数部分为1或这指数部分为0
6. 舍入，尾数可能超过52位，根据舍入模式对结果进行舍入
7. 检查，上溢则结果为∞，下溢则结果为0或非规格化数

### 乘法

1. 符号位异或
2. 指数位相加-1023
3. 尾数包含隐藏位直接相乘
4. 尾数相乘整数部分可能超过1，需要进行规格化
5. 舍入
6. 检查上溢下溢

#### 除法

1. 符号位异或
2. 指数位相减+1023
3. 尾数相除
4. 规格化
5. 舍入
6. 检查上溢下溢
